# Contributing

Contributions should improve the safe engineering, documentation, analysis, or simulation value of the project. The repository is intended for model-structure testing, sensor data analysis, and Blender comparison workflows.

## Contribution Rules

All contributions must remain within the project safety scope. Do not submit content that documents pyrotechnic acquisition, firework modification, ignition circuits, remote initiation, explosive placement, blast-charge design, or any other energetic-material procedure.

## Pull Request Checklist

| Requirement | Complete? |
|---|---:|
| The change is within the safe non-pyrotechnic scope. |  |
| The related issue is linked. |  |
| Documentation was updated where needed. |  |
| Example data or output is included if the change affects scripts. |  |
| Version or release notes were updated if applicable. |  |

## Review Standard

A reviewer should be able to understand what changed, why it changed, how it was tested, and whether it affects safety, data schema, code behavior, or simulation assumptions.
