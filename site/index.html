Six-Month Remote Engineering Project

Remote Structural Collapse Test Lab

A safe remote-first project for model-structure collapse testing, synchronized sensor capture, camera review, Python analytics, and Blender simulation.

Project Overview

This site organizes a two-collaborator engineering project through five gated phases. It provides the documentation, management workflow, data conventions, and safe code subprojects needed to build a reproducible measurement and simulation workflow.

Safety boundary: This project is scoped for non-pyrotechnic structural testing only. It does not include modified fireworks, explosive materials, ignition circuits, remote initiation, or blast-charge procedures.

Development Phases

Phase 1

Repository setup, bench data logging, safety checklist, and CSV/JSON schema.

Phase 2

Basic printed structures, weak-point geometry, and repeatable safe collapse tests.

Phase 3

Dual-camera views, timestamp validation, and remote review workflow.

Phase 4

Automated sync, Python analytics, plots, and Blender comparison notes.

Phase 5

Integrated model city demonstration, final data package, and release notes.
